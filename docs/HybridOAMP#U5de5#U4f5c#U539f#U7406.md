# HybridOAMP-Net 工作原理

**用最简单的方式理解模型是如何工作的**

---

## 核心思想：两步走

```
第1步：LMMSE初始化
   ↓
第2步：神经网络精炼（重复8次）
   ↓
输出结果
```

**生活比喻**：
- 第1步 = 用铅笔打草稿（快速、稳定）
- 第2步 = 用彩笔细修（慢慢优化）

---

## 第1步：LMMSE初始化

### 它做什么？

给你一个"还不错"的初步估计。

### 公式解释

**原始公式**（数学表达）：
```
x_初始 = (H* × H + 噪声系数 × I)^(-1) × H* × y
```

**符号说明**：
- `x_初始`：初步估计的发送符号
- `H`：信道矩阵
- `H*`：H的共轭转置（数学操作，不用深究）
- `I`：单位矩阵（对角线是1，其他是0）
- `y`：接收到的信号
- `噪声系数`：根据信噪比计算的一个数

**通俗理解**（不用记公式）：

想象你戴着有雾的眼镜看东西：

1. **算出眼镜怎么扭曲的**  
   → `H* × H`：信道的"能量"
   
2. **加一点容错**  
   → `+ 噪声系数 × I`：防止把噪声也放大
   
3. **反向操作去掉扭曲**  
   → `(...)^(-1) × H* × y`：用"逆操作"抵消信道影响

### 效果如何？

- ✅ 已经很接近真实值
- ✅ BER通常能到 6-7%
- ✅ 给深度学习一个好起点

## 第2步：神经网络精炼（8轮迭代）

### 基本流程

```
当前估计 x
   ↓
[每一轮精炼]
   ├─ 1) 计算还差多少
   ├─ 2) 往正确方向移动一点
   ├─ 3) 用神经网络去噪
   ↓
新的估计 x
```

重复8轮，每轮都比上一轮好一点点。

---

### 每一轮做什么？

#### 步骤1：计算还差多少

```
残差 = 接收信号 - 信道 × 当前估计
```

**比喻**：
- 你画了一条线
- 看看和目标线差多远
- 这个差距就是"残差"

#### 步骤2：计算修正方向

```
修正 = 当前估计 + 步长 × H* × 残差
```

**解释**：
- `H* × 残差`：把差距反向映射，算出"应该往哪改"
- `步长`：每次移动多少（类似学习率，初始值0.05）
- `当前估计 + 修正`：沿着正确方向走一小步

**比喻**：
```
画线时歪了：
1. 算出歪了多少（残差）
2. 算出应该往哪边改（修正方向）
3. 移动一小步（修正）
```

#### 步骤3：神经网络去噪

```
最终修正 = 去噪器(修正)
```

**去噪器是什么？**

一个很小的神经网络：
```
输入：复数符号（实部+虚部）
   ↓
第1层神经元（32个）
   ↓
激活函数ReLU
   ↓
第2层神经元（32个）
   ↓
激活函数ReLU
   ↓
输出：修正后的符号
```

**关键设计：残差连接**

```
输出 = 输入 + 网络的修正量
```

**比喻**：
- ❌ 不是"重新画一条线"
- ✅ 而是"在原来的线上微调"

**好处**：
- 即使网络学不好，至少保留原来的估计
- 只需学"微小的修正"，比学"完整的映射"容易得多

---

### 8轮累积效果

```
x₀ (LMMSE，BER≈7%)
 ↓ 第1轮精炼
x₁ (稍微好一点)
 ↓ 第2轮精炼
x₂ (继续进步)
 ↓ ...
 ↓ 第8轮精炼
x₈ (BER≈6%)
```

每轮提升一点点，8轮累加起来效果显著！

```
        │   第1步：LMMSE初始化  │

        ┌──────────────────────┐
        │   第2步：精炼（8轮）  │
        │                      │
        │  [第1轮]              │
        │   ① 计算残差          │
        │   ② 梯度更新          │
        │   ③ 去噪器 → x₁       │
        │                      │
        │  [第2轮]              │
        │   ① 计算残差          │
        │   ② 梯度更新          │
        │   ③ 去噪器 → x₂       │
        │                      │
        │  ... (重复到第8轮)    │
        │                      │
        │  [第8轮]              │
        │   ① 计算残差          │
        │   ② 梯度更新          │
        │   ③ 去噪器 → x₈       │
        └──────────┬───────────┘
                   ↓
        ┌──────────────────────┐
        │   输出：x₈ (最终估计) │
        └──────────────────────┘
```

### 去噪器结构（ResidualDenoiser）

```
输入：复数符号 (实部, 虚部)
   ↓
┌────────────────────────────┐
│  线性层1: 2 → 32           │  ← 从2维到32维
│  (把实部虚部映射到32个神经元) │
└─────────────┬──────────────┘
              ↓
┌────────────────────────────┐
│  ReLU激活                  │  ← 非线性变换
└─────────────┬──────────────┘
              ↓
┌────────────────────────────┐
│  线性层2: 32 → 32          │  ← 保持32维
└─────────────┬──────────────┘
              ↓
┌────────────────────────────┐
│  ReLU激活                  │
└─────────────┬──────────────┘
              ↓
┌────────────────────────────┐
│  线性层3: 32 → 2           │  ← 从32维回到2维
└─────────────┬──────────────┘
              ↓
┌────────────────────────────┐
│  残差连接:                  │
│  输出 = 输入 + 网络输出    │  ← 只学修正量
└─────────────┬──────────────┘
              ↓
输出：修正后的复数符号
```

**参数量**：
- 每个去噪器：约1200个参数
- 8个去噪器：约9600个参数
- 加上步长参数：约9800个参数（9.8K）

**对比**：
- ResNet（图像识别）：2500万参数
- GPT（语言模型）：1750亿参数
- HybridOAMP：**9800参数**（超轻量级！）

---

## 关键参数

| 参数 | 值 | 作用 |
|------|-----|------|
| 精炼层数 | 8 | 迭代修正的次数 |
| 步长γ | 0.05（初始） | 每次修正的幅度 |
| 神经元数 | 32 | 去噪器的隐藏层大小 |
| 总参数 | 9.8K | 整个模型的参数量 |

## 为什么这个设计有效？

### 原因1：好起点
LMMSE已经把BER降到7%，比随机初始化（50%）好太多。

**类比**：
- ❌ 困难：让小学生从零写高考作文
- ✅ 简单：让小学生把80分作文改到90分

### 原因2：残差学习
神经网络只需学"哪里不对，怎么修"，而不是"从零学整个检测"。

### 原因3：迭代优化
8轮累加，每轮提升一点点，最终效果显著。

### 原因4：参数少
只有9.8K参数，训练快（2-3分钟），不容易过拟合。

## 总结

### 核心公式（用文字表达）

```
第1步：
   x = LMMSE估计

第2步（重复8次）：
   ① 残差 = y - H·x
   ② 修正 = x + γ·H*·残差
   ③ x = 去噪器(修正)
```

### 记住这些就够了

1. **两步走**：LMMSE打基础 + 神经网络精炼
2. **8轮迭代**：每轮都比上一轮好一点
3. **残差学习**：只学修正量，不学全部
4. **超轻量**：9.8K参数，训练快
5. **效果好**：平均增益1.70 dB
