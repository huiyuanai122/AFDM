## 1. 物理信道模型

### 1.1 连续时间信道（论文公式3）

宽带双色散信道的输入输出关系：
```
y(t) = Σ_{i=1}^P h_i · x[(1+α_i)t - τ_i] + w(t)
```

**物理参数**：
- `h_i`: 第i条路径的复增益（考虑幅度衰减和相位偏移）
- `τ_i`: 第i条路径的时延（秒）
- `α_i`: 第i条路径的多普勒缩放因子（无量纲）

**物理意义**：
```
x[(1+α_i)t - τ_i]
    ↓
1. τ_i: 信号延迟τ_i秒到达
2. (1+α_i)t: 时间尺度变化
   - α_i > 0: 信号压缩（正多普勒，接近）
   - α_i < 0: 信号展宽（负多普勒，远离）
   - α_i = 0: 无多普勒，时间尺度不变
```

### 1.2 多普勒缩放因子与频率的关系

```
α_i = f_d,i / f_c
```
其中：
- `f_d,i = v_i / c × f_c`: 第i条路径的多普勒频移
- `v_i`: 相对速度
- `c`: 传播介质速度（水声：1500 m/s）
- `f_c`: 载波频率

**示例**（水声信道）：
```
f_c = 6 kHz
v = 0.15 m/s
c = 1500 m/s
→ f_d = 0.15/1500 × 6000 = 0.6 Hz
→ α = 0.6/6000 = 1e-4
```

## 2. 离散化与归一化

### 2.1 AFDM系统参数

```
N:  子载波数（例如 1024）
Δf: 子载波间隔（例如 4 Hz）
T:  符号周期 = N/Δf
B:  信号带宽 = N × Δf
```

### 2.2 物理参数 → 离散索引

#### 时延归一化
```
τ_i (秒) → l_i (离散索引)

l_i = ⌈τ_i × Δf × N⌉ = ⌈τ_i × B⌉
```

**示例**：
```
τ_i = 25 ms, Δf = 4 Hz, N = 1024
→ l_i = ⌈0.025 × 4 × 1024⌉ = 103
```

#### 多普勒归一化
```
α_i (物理) → α_i (离散索引)

α_i = ⌈f_d,i / Δf⌉
```

**示例**：
```
f_d,i = 0.6 Hz, Δf = 4 Hz
→ α_i = ⌈0.6 / 4⌉ = 1
```

### 2.3 参数范围

```
l_i ∈ [0, l_max]
α_i ∈ [-α_max, α_max]

其中：
l_max = ⌈τ_max × B⌉ = 103
α_max = ⌈(α_max,physical × f_c) / Δf⌉ = 1
```

## 3. DAF域等效信道推导

### 3.1 AFDM调制变换

**DAF域 → 时域（发射端）**：
```matlab
x_T[n] = Σ_{m=0}^{N-1} x[m] · exp(-j·2π·c1·m²) 
                       · exp(j·2πmn/N) 
                       · exp(-j·2π·c2·n²)
       = [A · x][n]
```

**矩阵形式**：
```matlab
F = dftmtx(N)/sqrt(N);
Lambda_c1 = diag(exp(-j·2π·c1·(0:N-1).²));
Lambda_c2 = diag(exp(-j·2π·c2·(0:N-1).²));
A = Lambda_c2 * F * Lambda_c1;
```

### 3.2 单径DAF域等效信道（论文公式19-27）

时域单径信道模型：
```
y_T[n] = h_i · x_T[(1+α_i)n - l_i] + w[n]
```

经过DAF变换后，DAF域等效信道矩阵Hi满足：
```
y = Hi · x + w'  （在DAF域）
```

### 3.3 核心公式（论文公式19-20）

```
Hi[p,q] = (1/N) · Σ_{n=0}^{N-1} exp{j·(2π/N)·Φ_i(p,q,n)}
```

**相位函数Φ_i(p,q,n)**：
```
Φ_i(p,q,n) = N·c2·(q² - p²) + n·(p - q - l_i) + α_i·n² - N·c1·l_i²
            |______________|   |_____________|   |______|   |__________|
                 ①                   ②            ③           ④
```

**各项物理意义**：
1. `N·c2·(q² - p²)`: AFDM chirp调制
2. `n·(p - q - l_i)`: **时延效应**（对应l_i）
3. `α_i·n²`: **时间尺度效应（多普勒）** ← 关键！
4. `-N·c1·l_i²`: 时延与chirp的交叉项

## 4. 单径信道矩阵Hi的计算

### 4.1 两种情况

#### 情况1：α_i = 0（无多普勒）

相位函数退化为：
```
Φ_i(p,q,n) = N·c2·(q² - p²) + n·(p - q - l_i) - N·c1·l_i²
            （没有n²项！）
```

可以**解析求解**：
```
Hi[p,q] = (1/N) · exp{j·2π·[c2·(q²-p²) - c1·l_i²]} 
                · Σ_{n=0}^{N-1} exp{j·2π·n·(p-q-l_i)/N}
```

利用几何级数：
```
Σ_{n=0}^{N-1} exp(j·2π·n·k/N) = N · δ(k mod N)
```

**结论**：只有当 `p - q - l_i ≡ 0 (mod N)` 时非零

即：`q = p + l_i (mod N)`

**代码实现**：
```matlab
if alpha_i == 0
    for p = 1:N
        q = mod(p + l_i - 1, N) + 1;
        phase = 2*pi * (c2 * (q^2 - p^2) - c1 * l_i^2);
        Hi(p, q) = exp(1j * phase);
    end
end
```

**矩阵结构**：单对角线，偏移l_i
```
l_i = 0  → 主对角线
l_i = 10 → 第10条对角线
l_i = 50 → 第50条对角线
```

#### 情况2：α_i ≠ 0（有多普勒）

相位函数包含`α_i·n²`二次项，需要POSP近似。

### 4.2 POSP近似

**驻点条件**：
```
dΦ/dn = p - q - l_i + 2·α_i·n = 0
→ n_sp = (q + l_i - p) / (2·α_i)
```

**驻点位置反推**：固定p，驻点对应的q：
```
q_sp = p + l_i·(1 - 2·N·c1)  （论文公式26）
```

**非零区域宽度**（论文公式34）：
```
Δq ≈ 2·N·c1·(2·|α_i| + 1 + 2·N_v/N)
```

### 4.3 计算流程（α_i ≠ 0）

**步骤1**：确定非零区域中心
```matlab
q_center = p + l_i*(1 - 2*N*c1);  % mod N
```

**步骤2**：确定非零区域宽度
```matlab
q_width = ceil(N*c1*(2*abs(alpha_i) + 1 + 2*Nv/N)) + 5;
```

**步骤3**：数值积分（只在非零区域）
```matlab
for q_raw = (q_center - q_width):(q_center + q_width)
    q = mod(q_raw - 1, N) + 1;
    phase_sum = 0;
    for n = 0:N-1
        theta = 2*pi/N * (...
            N * c2 * (q^2 - p^2) + ...
            n * (p - q - l_i) + ...
            alpha_i * n^2 + ...
            - N * c1 * l_i^2);
        phase_sum = phase_sum + exp(1j * theta);
    end
    Hi(p, q) = phase_sum / N;
end
```

**步骤4**：阈值处理
```matlab
threshold = 1e-2 * max(abs(Hi(:)));
Hi(abs(Hi) < threshold) = 0;
```

**作用**：去除数值积分旁瓣，保留主瓣。

### 4.4 三层循环结构详解 ⭐

**完整代码结构**：
```matlab
Hi = zeros(N, N);

% 外层循环：遍历所有输出子载波p
for p = 1:N  % ← 必须遍历所有p（1024次）
    
    % 计算稀疏区域中心（固定p后的驻点位置）
    q_center = p + l_i*(1 - 2*N*c1);
    
    % 计算稀疏区域宽度
    q_width = ceil(N*c1*(2*abs(alpha_i) + 1 + 2*Nv/N)) + 5;
    
    % 中层循环：只遍历稀疏区域的q
    for q_raw = (q_center - q_width):(q_center + q_width)  % ← 只遍历约20次！
        q = mod(q_raw - 1, N) + 1;
        
        phase_sum = 0;
        
        % 内层循环：数值积分计算Hi[p,q]
        for n = 0:N-1  % ← 1024次求和
            theta = 2*pi/N * (...
                N*c2*(q^2 - p^2) + ...
                n*(p - q - l_i) + ...
                alpha_i*n^2 + ...
                -N*c1*l_i^2);
            phase_sum = phase_sum + exp(1j*theta);
        end
        
        Hi(p, q) = phase_sum / N;
    end
end
```

#### 循环层次说明

**外层（p循环）**：
- **目的**：构建矩阵的每一行
- **范围**：p ∈ [1, N] = [1, 1024]
- **必要性**：✅ 必须遍历所有p，才能得到完整的N×N矩阵

**中层（q循环）**：
- **目的**：找到当前行p的非零元素
- **范围**：q ∈ [q_center - q_width, q_center + q_width] ≈ 20个
- **优化**：❌ 不遍历所有q（1024个），只遍历稀疏区域

**内层（n循环）**：
- **目的**：计算单个元素Hi[p,q]的数值积分
- **范围**：n ∈ [0, N-1] = [0, 1023]
- **必要性**：✅ 完整的DFT求和

#### 为什么q_center随p变化？

**驻点条件**：
```
dΦ/dn = p - q - l_i + 2·α_i·n = 0
→ n_sp = (q + l_i - p) / (2·α_i)
```

**反推q_sp（驻点对应的q）**：
```
q_sp = p + l_i·(1 - 2·N·c1)
```

**关键发现**：
- q_sp是p的**线性函数**
- **不同的p → 不同的q_sp**
- → 矩阵呈**对角带状结构**

**数值示例**：
```
假设 l_i = 50, N = 1024, c1 = 0.00146

p = 1:    q_center = 1 + 50×(-1.99) ≈ -98 → mod得到 927
p = 2:    q_center = 2 + 50×(-1.99) ≈ -97 → mod得到 928
p = 3:    q_center = 3 + 50×(-1.99) ≈ -96 → mod得到 929
...
p = 1024: q_center = 1024 + 50×(-1.99) ≈ 925

→ 对角带斜率 ≈ 1 + 50×(-2×1024×0.00146) ≈ -2.0
```

### 4.5 计算复杂度分析

#### 朴素方法（遍历所有q）

```
外层循环: N次（p）
中层循环: N次（q）← 遍历所有q
内层循环: N次（n）
→ 总复杂度: O(N³) = O(1024³) ≈ 10⁹次运算
```

#### POSP优化方法（只遍历稀疏q）

```
外层循环: N次（p）
中层循环: 2×q_width次（q）≈ 20次 ← 只遍历稀疏区域
内层循环: N次（n）
→ 总复杂度: O(N × q_width × N) = O(N² × 20) ≈ 2×10⁷次
→ 节省 98% 计算量！
```

#### 对比表

| 方法 | 复杂度 | N=1024时 | 相对速度 |
|-----|-------|---------|---------|
| 朴素 | O(N³) | 10⁹次 | 1× |
| POSP | O(N²·q_width) | 2×10⁷次 | **50×** ✓ |

**结论**：POSP近似不仅理论上成立，而且大幅提升计算效率！
