## 5. 多径信道矩阵的叠加

### 5.1 多径叠加

完整的DAF域信道矩阵：
```
H = Σ_{i=1}^P h_i · Hi
```

其中：
- `P`: 路径数（例如4）
- `h_i`: 第i条路径的复增益（Rayleigh衰落）
- `Hi`: 第i条路径的单径信道矩阵

**代码实现**：
```matlab
Heff = zeros(N, N);
for i = 1:num_paths
    Hi = build_Hi_correct(N, c1, c2, l_i(i), alpha_i(i));
    Heff = Heff + h_true(i) * Hi;  % 加权叠加
end
```

### 5.2 功率归一化

确保SNR定义准确：
```matlab
H_eff_sub = Heff(Q+1:N-Q, Q+1:N-Q);  % 有效子空间
sub_power = norm(H_eff_sub, 'fro')^2 / N_eff;
Heff = Heff / sqrt(sub_power);  % 归一化
```

确保：`E[|H·x|²] ≈ E[|x|²] = 1`

## 6. 完整代码解析

### 6.1 函数签名

```matlab
function Hi = build_Hi_correct(N, c1, c2, l_i, alpha_i)
```

**输入**：
- `N`: 子载波数（1024）
- `c1, c2`: AFDM chirp参数
- `l_i`: 当前路径时延索引（0~103）
- `alpha_i`: 当前路径多普勒索引（-1~1）

**输出**：
- `Hi`: N×N单径信道矩阵

### 6.2 完整代码（test_detectors_comparison.m 第551-630行）

```matlab
function Hi = build_Hi_correct(N, c1, c2, l_i, alpha_i)
    Hi = zeros(N, N);
    Nv = 2;
    
    if alpha_i == 0
        %% ===== α_i = 0: 解析解 =====
        for p = 1:N
            q = mod(p + l_i - 1, N) + 1;
            phase = 2*pi * (c2 * (q^2 - p^2) - c1 * l_i^2);
            Hi(p, q) = exp(1j * phase);
        end
        
    else
        %% ===== α_i ≠ 0: POSP近似 + 数值积分 =====
        for p = 1:N
            % 驻点中心
            q_center_float = p + l_i*(1 - 2*N*c1);
            q_center = mod(round(q_center_float) - 1, N) + 1;
            
            % 非零区域宽度（关键修正！）
            q_width = ceil(N * c1 * (2*abs(alpha_i) + 1 + 2*Nv/N)) + 5;
            
            % 只在非零区域计算
            for q_raw = (q_center - q_width):(q_center + q_width)
                q = mod(q_raw - 1, N) + 1;
                phase_sum = 0;
                
                % 数值积分
                for n = 0:N-1
                    theta = 2*pi/N * (...
                        N * c2 * (q^2 - p^2) + ...  % ① Chirp
                        n * (p - q - l_i) + ...      % ② 时延
                        alpha_i * n^2 + ...          % ③ 多普勒
                        - N * c1 * l_i^2 ...         % ④ 交叉项
                    );
                    phase_sum = phase_sum + exp(1j * theta);
                end
                
                Hi(p, q) = phase_sum / N;
            end
        end
    end
    
    %% 阈值处理
    threshold = 1e-2 * max(abs(Hi(:)));
    if threshold > 0
        Hi(abs(Hi) < threshold) = 0;
    end
end
```

## 8. 参数到矩阵的完整映射

### 8.1 映射链

```
物理参数 → 离散参数 → 矩阵特征 → 代码位置
─────────────────────────────────────────────
τ_i (秒) → l_i (索引) → 对角线偏移 → 第576, 591行

h_i (复数) → h_i (复数) → 矩阵幅度   → 第196行

### 8.2 直观理解

**时延l_i的作用**：
```
l_i = 0   → Hi[p,q]在q≈p附近非零（主对角线）
l_i = 25  → Hi[p,q]在q≈p+25附近非零（第25条对角线）
l_i = 50  → Hi[p,q]在q≈p+50附近非零（第50条对角线）
l_i = 75  → Hi[p,q]在q≈p+75附近非零（第75条对角线）
```

**多普勒α_i的作用**：
```
α_i = 0   → 单对角线，每行1个非零元素
α_i = ±1  → 对角带，每行约20个非零元素
α_i = ±5  → 宽带，每行约100个非零元素
```

**复增益h_i的作用**：
```
|h_i| = 0.6  → Hi整体缩放为原来的0.6倍
∠h_i = π/4   → Hi所有元素乘以exp(j·π/4)
```

### 8.3 多径叠加示例

```
4径信道：
- 路径1: l_1=0,  α_1=-1, h_1=0.5   → H_1在主对角线
- 路径2: l_2=26, α_2=-1, h_2=0.3   → H_2在第26条对角线
- 路径3: l_3=51, α_3=1,  h_3=0.2   → H_3在第51条对角线
- 路径4: l_4=77, α_4=1,  h_4=0.1   → H_4在第77条对角线

## 9. 关键技术细节

### 9.1 为什么α_i = 0需要特殊处理？

**原因**：
1. 相位函数退化为线性，POSP近似不适用
2. 可以利用DFT性质解析求解
3. 更准确、更快（无需数值积分）

### 9.2 为什么阈值是1e-2？

**数值积分的问题**：
```
理论（POSP）: 只有主瓣
数值积分:     主瓣 + 旁瓣（数值误差）

旁瓣幅度通常 < 主瓣的1%（-40dB）
```

**阈值选择**：
- `1e-6`：太小，保留旁瓣 → Hi不够稀疏
- `1e-2`：合适，去除旁瓣 → Hi稀疏如论文
- `1e-1`：太大，损失主瓣 → 信道失真

### 9.3 为什么使用abs(alpha_i)？

**错误做法**（之前）：
```matlab
q_width = ceil(N * c1 * (2*alpha_max + ...));  % 用alpha_max
```
问题：所有路径用相同带宽，alpha_i小的路径会高估

**正确做法**（现在）：
```matlab
q_width = ceil(N * c1 * (2*abs(alpha_i) + ...));  % 用abs(alpha_i)
```
优点：每条路径根据自己的多普勒计算准确带宽

### 9.4 为什么q_width要加5？

**理论带宽 vs 实际带宽**：
```
理论: Δq = 2·N·c1·(2·|α_i| + 1 + 2·N_v/N)
实际: 驻点附近可能略超出理论值

+5: 裕量，防止边界截断
```

### 9.5 参数自动计算 vs 硬编码 ⭐

**问题**：早期代码中α_max是硬编码的
```matlab
alpha_max = 1;  % ❌ 硬编码，不灵活
```

**缺点**：
- 改变fc或df时，需要手动更新α_max
- 不清楚1是怎么来的
- 与l_max的计算方式不一致

**改进**：根据物理参数自动计算 ✓
```matlab
% 物理参数
alpha_max_paper = 1e-4;            % 物理多普勒因子
fd_max = alpha_max_paper * fc;     % 最大多普勒频移 (Hz)

% 归一化参数（自动计算）
alpha_max = ceil(fd_max / df);     % ✓ 归一化索引
l_max = ceil(tau_max * fs);        % ✓ 归一化索引
```

**优点**：
- ✅ 参数改变时自动更新
- ✅ 计算过程清晰透明
- ✅ 与l_max保持一致的风格
- ✅ 便于参数扫描和优化

**代码位置**：
- 多符号脚本：`test_frame_daf_domain.m` 第45-46行
- 单符号脚本：`test_detectors_comparison.m` 第67行

## 10. 常见错误与调试

### 10.1 错误1：信道矩阵全零

**症状**：
```matlab
norm(Hi, 'fro') = 0
```

**原因**：
- `alpha_i = 0`时忘记特殊处理
- 阈值设置过大

**检查**：
```matlab
fprintf('Max |Hi| before threshold: %.6e\n', max(abs(Hi(:))));
fprintf('Threshold: %.6e\n', threshold);
```

### 10.2 错误2：ICI过高

**症状**：
```
接收信号功率 >> 发射信号功率
BER在高SNR下仍然很高
```

**原因**：
- 阈值太小，保留了旁瓣
- `q_width`计算错误（用了`alpha_max`而非`abs(alpha_i)`）

**检查**：
```matlab
nnz_ratio = nnz(Hi) / N^2 * 100;
fprintf('Hi稀疏度: %.2f%% 非零元素\n', nnz_ratio);
% 期望值: α_i=0时约0.1%, α_i=1时约2-5%
```

### 10.3 错误3：功率不匹配

**症状**：
```
mean(abs(y).^2) 远离 1
```

**原因**：
- 发射端归一化错误
- 信道归一化错误

**检查**：
```matlab
fprintf('发射功率: %.4f\n', mean(abs(x).^2));
fprintf('信道Frobenius范数²/N: %.4f\n', norm(Heff,'fro')^2/N);
fprintf('接收功率: %.4f\n', mean(abs(y).^2));
```

## 11. 总结

### 11.1 核心公式

```
Hi[p,q] = (1/N) · Σ_{n=0}^{N-1} exp{j·(2π/N)·Φ_i(p,q,n)}

Φ_i(p,q,n) = N·c2·(q²-p²) + n·(p-q-l_i) + α_i·n² - N·c1·l_i²
            └─ Chirp ─┘   └─ 时延 ─┘   └多普勒┘  └交叉项┘
```

### 11.2 参数映射表

| 物理参数 | 值示例 | 离散参数 | 矩阵影响 |
|---------|-------|---------|---------|
| τ_i | 25 ms | l_i = 103 | 对角线偏移103 |
| α_i | 1e-4 | α_i = 1 | 带宽≈20子载波 |
| h_i | 0.6+0.3j | h_i | 幅度0.6，相位∠0.3 |

### 11.3 实现要点

1. ✓ **α_i = 0特殊处理**：解析解，单对角线
2. ✓ **POSP近似**：驻点中心 + 有限带宽
3. ✓ **abs(alpha_i)**：每条路径独立计算带宽
4. ✓ **阈值1e-2**：去除旁瓣，保持稀疏性
5. ✓ **多径叠加**：加权求和 + 功率归一化

### 11.4 三层循环可视化理解 ⭐

**问题**：是否要遍历所有p？
**答案**：✅ 是的！必须遍历所有p才能得到完整矩阵。

**完整流程**：
```
┌────────────────────────────────────────────┐
│ 外层：for p = 1:N  (遍历所有行)              │
│   ↓                                         │
│   计算 q_center(p) ← 对每个p计算驻点中心    │
│   计算 q_width     ← 非零区域宽度           │
│   ↓                                         │
│   ┌──────────────────────────────────────┐ │
│   │ 中层：for q in 稀疏区域  (~20次)     │ │
│   │   ↓                                   │ │
│   │   ┌────────────────────────────────┐ │ │
│   │   │ 内层：for n = 0:N-1  (1024次) │ │ │
│   │   │   计算 exp{j·Φ(p,q,n)}        │ │ │
│   │   │   累加到 phase_sum             │ │ │
│   │   └────────────────────────────────┘ │ │
│   │   Hi[p,q] = phase_sum / N           │ │
│   └──────────────────────────────────────┘ │
└────────────────────────────────────────────┘
```

**关键点**：
1. ✅ **p必须全遍历**（1024次）→ 得到完整矩阵
2. ✅ **q只遍历稀疏区域**（~20次）→ POSP优化
3. ✅ **n必须全求和**（1024次）→ DFT积分
4. ✅ **q_center随p线性变化** → 对角带结构

**矩阵构建过程可视化**：
```
       q →
   ┌───────────────┐
 p │ · · · · · · · │ ← p=1: 计算q∈[920,940]的20个元素
 ↓ │ · · · · · · · │ ← p=2: 计算q∈[921,941]的20个元素
   │ · · · · · · · │ ← p=3: 计算q∈[922,942]的20个元素
   │      ...      │
   │ · · · · · · · │ ← p=1024: 计算q∈[917,937]的20个元素
   └───────────────┘

每行只计算约20个非零元素，而非1024个！
```

### 11.5 从物理到代码的完整流程

```
1. 物理信道 y(t) = Σ h_i·x[(1+α_i)t - τ_i]
   ↓ 采样 + 归一化
2. 离散参数 {l_i, α_i, h_i}
   归一化公式：
   - l_i = ⌈τ_i × fs⌉ ∈ [0, l_max]
   - α_i = ⌈f_d,i / Δf⌉ ∈ [-α_max, α_max]
   ↓ DAF变换
3. 等效信道 y = H·x, H = Σ h_i·Hi
   ↓ build_Hi_correct(N, c1, c2, l_i, α_i)
4. 单径矩阵 Hi[p,q] = (1/N)·Σ_n exp{j·Φ(p,q,n)}
   ↓ 三层循环：p全遍历 + q稀疏遍历 + n求和
5. 稀疏对角带状矩阵
   - α_i=0: 单对角线（0.1%非零）
   - α_i≠0: 对角带（2-5%非零）
```
